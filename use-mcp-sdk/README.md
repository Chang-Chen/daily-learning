### 🪄 MCP Server 的可视化界面调试工具

```npm
npx @modelcontextprotocol/inspector
```

### 🪄 MCP Server 的SDK

```npm
npm install @modelcontextprotocol/sdk
```

```shell

# 第一步：安装依赖

cd use-mcp-sdk && pnpm i

# 第二步：启动可视化服务

npx @modelcontextprotocol/inspector

# 第三步：界面配置
# Transport Type：选择 STDIO 协议
# Command：node
# Arguments：src/server.js
# 点击 Connect 链接 MCP  服务
```
